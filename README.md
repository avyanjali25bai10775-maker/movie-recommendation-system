# movie-recommendation-system
a simple python program that gives you movie recommendations similar to your entered movie name

# 🎬 Movie Recommendation System

## 📌 Problem
Finding similar movies manually is time-consuming and inefficient.

## 💡 Solution
This project uses a content-based filtering approach to recommend movies based on similarity.

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn

## 🚀 How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run the program:
   python app.py

3. Enter a movie name

## 🧠 Concept Used
Cosine Similarity

## 📊 Dataset
TMDB Movie Dataset

## 📌 Output
Recommends top 5 similar movies
